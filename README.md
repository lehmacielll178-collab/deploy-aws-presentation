# Apresentação para Deploy na AWS

Este é o código de uma aplicação gerada com IA para uma apresentação de como fazer deploy na AWS. A aplicação visa ser um CRUD simples e não é o foco em questão, por isso ela foi gerada com IA e questões como segurança da aplicação em si foram intencionalmente negligenciadas de forma que a aplicação fosse o mais simples possível de fazer deploy.

Não utilize o esse código para fazer um deploy real pois ele está cheio de VULNERABILIDADES intencionalmente não resolvidas
